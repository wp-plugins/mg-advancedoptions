With this Plugin you can activate or deactivate the WordPress post revision system and the Autosave-Functions over the Adminpanel.

== Description ==

The new post revision system in Wordpress 2.6 needs a lot of database performance and with this Plugin these can be turned off. The Plugin can managed comfortable over the Adminpanel.

== Installation ==

1. Download the Pluginfile
2. Unzip the File on your local PC
3. Copy `MG_AdvancedOptions.php` in your Wordpress Plugin Directory
4. Activate the Plugin through the 'Plugins' menu in WordPress
5. Manage your Settings here: *WP Admin -> Settings -> AdvancedOptions*
